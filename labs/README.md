# Labs

One folder per lab. Each lab is a complete artifact proving competence — and the source of one video.

## Naming
`NNNN-short-slug/` — e.g. `0001-ospf-single-area/`, `0012-aws-vpc-hybrid/`.

## Each lab folder contains
- `README.md` — objective, topology, steps, what I learned, gotchas
- `diagram.*` — the network/architecture diagram
- `configs/` — device configs / IaC / scripts
- `video.md` — link to the published video + the script/talking points
