# Mission — The Charter

## 3-Year Career Mission (2026–2029)

Become a highly respected **customer-facing AWS Solutions Architect** while building the
technical depth of a **Network Automation Engineer**.

- **Target date:** June 30, 2029
- **Financial goal:** $200,000+ annual compensation (by 2030)
- **Current position:** AWS Infrastructure Delivery Technician (L4)
- **Expected progression:** Move toward management in 12–24 months while continuing
  technical development outside of work.

---

## Guiding Principles

Every recommendation must pass these filters:

1. Does this move me toward becoming a Solutions Architect?
2. Does this increase my value as a Network Automation Engineer?
3. Does this improve my communication ability?
4. Will this improve my earning potential?
5. Is this the highest Return on Investment for my time?

If the answer is **No** → postpone it until after June 2029. No shiny objects. Protect focus.

---

## Core Career Identity

I am becoming:
- Enterprise Networking Professional
- Cloud Architect
- Infrastructure Automation Engineer
- Technical Communicator
- Technical Leader

**The synthesis:** the rare Solutions Architect who deeply understands enterprise networking.
Everything I study must reinforce one or more of these identities.

---

## Philosophy

- Certifications are **not** the goal. **Knowledge** is the goal.
- **Projects prove competence.** Certifications provide recognition and get résumés interviews.
- Always prioritize **understanding over memorization**.
- **Teaching is part of studying.** If I can't explain it clearly, I don't understand it well enough.
- No certification exists without a supporting project (repo + docs + diagram + write-up + video).

---

## Communication Goal

- Build one lab → study it → understand it deeply → teach it.
- **~52 educational videos per year** (target), holiday-aware cadence (see ROADMAP).
- Over two years: **100+ videos**.
- Audience: aspiring Infrastructure, Network, and Cloud Engineers.
- Explain simply — diagrams, labs, analogies, real-world examples.

---

## The 3-Year Arc (Jul 2026 → Jun 2029)

- **Year 1 — Enterprise Networking:** CCNP Enterprise (ENCOR + ENARSI). Git mastered. ~52 videos, ~25 labs.
- **Year 2 — Cloud + Automation:** AWS SA Associate. Ansible (deep skill) + Terraform/IaC + GitHub Actions.
  Automation capstone. ~104 videos total.
- **Year 3 — Specialization + the role:** AWS Advanced Networking Specialty (the niche credential).
  Flagship hybrid capstone. Land the customer-facing SA role. ~150+ videos, 50+ labs total.

_(Optional Red Hat Ansible cert decided in Year 3 on market signal. Small course corrections expected —
this is the baseline.)_
