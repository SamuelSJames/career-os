# Roadmap — 3-Year Mission (Jul 1, 2026 → Jun 30, 2029)

Toward **Solutions Architect by June 30, 2029** and **$200K+ by 2030**.

**Strategy: sequence, don't parallelize. Clean theme per year** so each goes deep without
context-thrash, and the pace stays sustainable for a full-time worker:
- **Year 1 = Enterprise Networking + CCNP** (finish the networking credential)
- **Year 2 = Cloud foundation + Automation skill** (bring networking to the cloud; learn Ansible/IaC deep)
- **Year 3 = Specialization + landing the SA role** (AWS Advanced Networking Specialty + capstone)

Git & GitHub are learned *as the tooling that runs the labs* — not separate study blocks.

**Video cadence:** ~52/year → **~150+ over three years** (charter asked 100+; we'll exceed it). Off
**Thanksgiving → Christmas (~5 weeks) EVERY year.** ~47 working weeks/yr → bank a buffer (~2/wk)
Oct–mid Nov and schedule-publish through the break.

**Cert summary by year:**
| Year | Theme | Certs |
|---|---|---|
| Year 1 | Enterprise Networking | CCNP ENCOR + ENARSI → **CCNP complete** |
| Year 2 | Cloud + Automation | **AWS SA Associate** + GitHub Actions/IaC + **Ansible (deep skill)** |
| Year 3 | Specialization + SA role | **AWS Advanced Networking – Specialty (ANS-C01)** + capstone + *optional* Ansible cert |

# YEAR 1 — Jul 2026 → Jun 2027 · Enterprise Networking + CCNP

Finish **CCNP Enterprise** (ENCOR core + ENARSI concentration). Build the networking foundation
that makes me the rare SA who deeply understands enterprise networks.

### Q1 — Jul–Sep 2026 · CCNP ENCOR Core (the spine)
- **Study:** routing fundamentals, OSPF (single/multi-area), EIGRP basics, VLANs, STP, EtherChannel.
- **Labs:** ~1/week in pnetlab (OSPF, STP, EtherChannel, inter-VLAN, HA basics).
- **Video:** ~1/week (≈13). Establish the filming/editing workflow early.
- **Tooling:** Git workflow + repo-per-lab habit; learn branching/PRs by using them.

### Q2 — Oct–Dec 2026 · ENCOR continued + BANK BUFFER 🦃🎄
- **Study (Oct–mid Nov):** BGP, route maps, redistribution, HA (HSRP/VRRP), wireless,
  enterprise architecture, security, automation concepts.
- **Labs:** ~1–2/week (BGP/HA = great visual content).
- **Video (Oct–mid Nov):** ~2/week, bank ~6–8 before Thanksgiving.
- **🛑 Thanksgiving → Christmas: OFF.** Banked videos auto-publish. Accountability pauses.
- **Target:** sit **ENCOR 350-401 ~Feb 2027** (study carries into early Q3).

### Q3 — Jan–Mar 2027 · ENCOR exam + start ENARSI
- **Jan–Feb:** final ENCOR review + practice exams → **pass ENCOR.**
- **Feb–Mar:** begin **ENARSI 300-410** — EIGRP/OSPF/BGP deep, VPN services (DMVPN),
  infrastructure security & services, troubleshooting.
- **Labs + Video:** ~1/week (≈13).

### Q4 — Apr–Jun 2027 · Finish ENARSI → CCNP COMPLETE
- **Study:** complete ENARSI; heavy troubleshooting labs.
- **Target:** **pass ENARSI → CCNP Enterprise complete by ~Jun 2027.** ✅
- **Labs + Video:** ~1/week (≈12). Consolidate the networking portfolio.

**End of Year 1:** CCNP done · ~52 videos · ~25 labs · Git mastered · architect-level networking.
