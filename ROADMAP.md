# Roadmap — Year One (Jul 1, 2026 → Jun 30, 2027)

**Strategy: sequence, don't parallelize.** One big rock at a time. Git & GitHub Actions are
learned *as the tooling that runs the labs*, not as separate study blocks — highest ROI.

**Video cadence:** target 52/year. Off **Thanksgiving → Christmas (~5 weeks)**. That leaves
~47 working weeks, so real pace is ~1.1/week → **bank a buffer (2/week) before the break** and
schedule-publish through it. The channel stays alive while I rest.

---

## Quarter Plan

### Q1 — Jul–Sep 2026 · CCNP ENCOR Core (the spine)
- **Study:** routing fundamentals, OSPF (single/multi-area), EIGRP basics, VLANs, STP, EtherChannel.
- **Labs:** ~1/week (OSPF, STP, EtherChannel, inter-VLAN routing, HA basics).
- **Video:** ~1/week (≈13). Establish the filming/editing workflow early — speed comes from reps.
- **Tooling:** set up Git workflow + repo-per-lab habit. Learn branching/PRs by using them.

### Q2 — Oct–Dec 2026 · ENCOR continued + BANK BUFFER 🦃🎄
- **Study (Oct–mid Nov):** BGP, route maps, redistribution, high availability (HSRP/VRRP),
  wireless fundamentals, enterprise architecture, security, automation concepts.
- **Labs:** ~1–2/week — rich, visual BGP/HA material = great video content.
- **Video (Oct–mid Nov):** **~2/week** to bank ~6–8 videos before Thanksgiving.
- **🛑 Thanksgiving → Christmas: OFF.** Produce nothing; banked videos auto-publish.
  Accountability check pauses (no guilt). Brain consolidates before Q3.

### Q3 — Jan–Mar 2027 · AWS Solutions Architect Associate
- **Study:** VPC design, hybrid networking, IAM, EC2, ELB, Route 53, S3, security, cost
  optimization, HA, disaster recovery. Focus on **design thinking** — explain *why*.
- **Labs:** ~1/week (build real VPCs, hybrid connectivity, HA patterns in AWS).
- **Video:** ~1/week (≈13). Networking-in-AWS angle = my differentiator on camera.

### Q4 — Apr–Jun 2027 · CCNP Concentration + Consolidation
- **Study:** chosen CCNP concentration (e.g. ENARSI or automation-focused) → complete CCNP.
- **Labs + Video:** ~1/week (≈12). Consolidate, revisit weak areas, polish the portfolio.
- **GitHub Actions:** formalize CI/CD understanding by automating lab validation / repo workflows.
