# Roadmap — 2-Year Foundation (Jul 1, 2026 → Jun 30, 2028)

Toward the full mission target of **June 30, 2029** (Solutions Architect) and **$200K+ by 2030**.

**Strategy: sequence, don't parallelize. Clean domain split across two years** so each goes deep
without context-thrash, and the pace stays sustainable for a full-time worker:
- **Year 1 = Enterprise Networking + CCNP** (finish the networking credential)
- **Year 2 = Cloud + Automation + Communication depth** (bring networking to the cloud)

Git & GitHub are learned *as the tooling that runs the labs* — not separate study blocks.

**Video cadence:** ~52/year → **104+ over two years** (matches the charter's 100+). Off
**Thanksgiving → Christmas (~5 weeks)** BOTH years. ~47 working weeks/yr → ~1.1/wk, so
**bank a buffer (~2/wk) Oct–mid Nov** and schedule-publish through the break. Channel stays alive.

# YEAR 1 — Jul 2026 → Jun 2027 · Enterprise Networking + CCNP

Finish **CCNP Enterprise** (ENCOR core + ENARSI concentration). Build the networking foundation
that makes me the rare SA who deeply understands enterprise networks.

### Q1 — Jul–Sep 2026 · CCNP ENCOR Core (the spine)
- **Study:** routing fundamentals, OSPF (single/multi-area), EIGRP basics, VLANs, STP, EtherChannel.
- **Labs:** ~1/week in pnetlab (OSPF, STP, EtherChannel, inter-VLAN, HA basics).
- **Video:** ~1/week (≈13). Establish the filming/editing workflow early.
- **Tooling:** Git workflow + repo-per-lab habit; learn branching/PRs by using them.

### Q2 — Oct–Dec 2026 · ENCOR continued + BANK BUFFER 🦃🎄
- **Study (Oct–mid Nov):** BGP, route maps, redistribution, HA (HSRP/VRRP), wireless,
  enterprise architecture, security, automation concepts.
- **Labs:** ~1–2/week (BGP/HA = great visual content).
- **Video (Oct–mid Nov):** ~2/week, bank ~6–8 before Thanksgiving.
- **🛑 Thanksgiving → Christmas: OFF.** Banked videos auto-publish. Accountability pauses.
- **Target:** sit **ENCOR 350-401 ~Feb 2027** (study carries into early Q3).

### Q3 — Jan–Mar 2027 · ENCOR exam + start ENARSI
- **Jan–Feb:** final ENCOR review + practice exams → **pass ENCOR.**
- **Feb–Mar:** begin **ENARSI 300-410** (advanced routing) — EIGRP/OSPF/BGP deep, VPN
  services (DMVPN), infrastructure security & services, troubleshooting.
- **Labs + Video:** ~1/week (≈13).

### Q4 — Apr–Jun 2027 · Finish ENARSI → CCNP COMPLETE
- **Study:** complete ENARSI; heavy troubleshooting labs.
- **Target:** **pass ENARSI → CCNP Enterprise complete by ~Jun 2027.** ✅
- **Labs + Video:** ~1/week (≈12). Consolidate the networking portfolio.

**End of Year 1:** CCNP Enterprise done · ~52 videos · ~25 labs · Git mastered · architect-level
networking understanding.

# YEAR 2 — Jul 2027 → Jun 2028 · Cloud + Automation + Communication

Bring the networking depth into AWS. Earn the SA Associate, formalize automation/CI-CD, and
sharpen the technical-communicator brand.

### Q1 — Jul–Sep 2027 · AWS Solutions Architect Associate
- **Study:** VPC design, hybrid networking (VPN/Direct Connect/Transit Gateway), IAM, EC2, ELB,
  Route 53, S3, security, cost optimization, HA, disaster recovery. Focus: **design thinking — why.**
- **Labs:** ~1/week — real VPCs, hybrid connectivity, HA patterns. Lean on the networking edge.
- **Video:** ~1/week (≈13). "Networking-in-AWS" angle = my differentiator.

### Q2 — Oct–Dec 2027 · SAA exam + BANK BUFFER 🦃🎄
- **Oct–Nov:** final SAA review + practice → **pass AWS SA Associate.**
- **Video (Oct–mid Nov):** ~2/week, bank for the break.
- **🛑 Thanksgiving → Christmas: OFF.** Publish from bank; accountability pauses.

### Q3 — Jan–Mar 2028 · Infrastructure Automation + IaC

  Terraform/IaC for network + cloud, Python network automation.
- **Labs:** ~1/week — automate lab provisioning, CI-validate configs, IaC deployments.
- **Video:** ~1/week (≈13). Automation content reinforces the Network Automation Engineer identity.

### Q4 — Apr–Jun 2028 · Capstone + Consolidation
- **Build:** a flagship multi-domain project — hybrid enterprise network + AWS, fully IaC,
  CI/CD-driven, documented with architecture diagrams. The portfolio centerpiece.
- **Labs + Video:** ~1/week (≈12). Polish repos, write-ups, and the channel.

**End of Year 2:** AWS SAA done · GitHub Actions/IaC mastered · ~104 videos total · ~50 labs total ·
flagship capstone · thinking like an architect across networking AND cloud.
